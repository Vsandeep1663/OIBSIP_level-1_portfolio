Brick Breaker Using Python

Brick Breaker is a breakout clone in which the player must smash a wall of bricks by deflecting a bouncing ball with a paddle. The paddle may move horizontally and it is controlled by user using key board or mouse. The player gets 3 lives to start with a life is lost if the ball hits the bottom of the screen. When all bricks are destroyed the player is the winner of the game . If all lives are lost , the game is over.
